## Overview

The Ecommerce Management System is a backend application developed using Java and Spring Core Framework. The system simulates core e-commerce operations and demonstrates enterprise-level dependency injection and IoC concepts.

The project is designed to reflect real-world backend architecture and is intended for portfolio presentation, internship applications, and technical interview preparation.

## Spring Core Concepts Implemented

### Inversion of Control (IoC)
The Spring container manages object creation and dependency injection instead of manual instantiation.

### Dependency Injection
Constructor-based dependency injection is used to inject required services into classes.

### BeanFactory and ApplicationContext
Demonstrates both BeanFactory (basic container) and ApplicationContext (advanced container).

### XML Configuration
Spring beans are configured using XML in early stages of the project.

### Spring Annotations
Annotations such as @Component, @Service, @Repository, @Autowired are used for configuration.

### Autowiring
Dependencies are automatically injected by Spring using autowiring mechanism.

### Bean Lifecycle
Spring bean lifecycle is demonstrated using @PostConstruct and @PreDestroy.

### Bean Scope
Singleton and Prototype scopes are implemented and compared.
